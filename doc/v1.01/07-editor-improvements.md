# 编辑器改进

## 变更概述

修复 Tab 键和 Markdown 工具栏两个问题。

## 变更文件

| 文件 | 变更内容 |
|------|----------|
| `src/renderer/components/editor/MdEditor.vue` | Tab 键插入空格；工具栏在光标位置插入标记；选中文本包裹标记 |

## 1. Tab 键修复

### 问题

在 textarea 中按 Tab 键，焦点会切换到其他按钮，无法输入制表符。

### 修复

添加 `@keydown.tab.prevent="handleTab"`，使用 `document.execCommand('insertText', false, '  ')` 插入两个空格。

选择 `execCommand` 而非直接赋值 `editContent.value`，是因为 `execCommand` 会将操作记录到浏览器撤销栈，`Ctrl+Z` 可以正常撤销。

## 2. Markdown 工具栏修复

### 问题

点击加粗、斜体、标题等工具栏按钮，标记只会追加到文章末尾，而非在光标位置插入。

### 修复

重写 `insertMarkdown` 函数：

- **无选中文本**：在光标位置插入标记，光标定位到合适位置（如加粗后光标在 `**|**` 中间）
- **有选中文本**：在选中文本两侧包裹标记（如选中"文字"后点加粗 → `**文字**`）
- 使用 `document.execCommand('insertText')` 插入，支持 `Ctrl+Z` 撤销

### 支持的语法

| 语法 | 无选中 | 有选中 |
|------|--------|--------|
| 加粗 | `**\|**` | `**选中**` |
| 斜体 | `*\|*` | `*选中*` |
| 行内代码 | `` `\|` `` | `` `选中` `` |
| H1/H2/H3 | 行首插入 `# ` | 行首插入 `# 选中` |
| 引用 | 行首插入 `> ` | 行首插入 `> 选中` |
| 列表 | 行首插入 `- ` | 行首插入 `- 选中` |
| 链接 | `[\|]()` | `[选中]()` |
