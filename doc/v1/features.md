# NoteMaster 功能模块

## 核心功能清单

### 1. 笔记管理
- Markdown 编辑（markdown-it 渲染 + highlight.js 代码高亮）
- 三种编辑模式：纯编辑 / 纯预览 / 分栏（同步滚动）
- 自动保存（2 秒防抖）
- 笔记置顶、收藏
- 笔记历史版本（每次保存自动快照）
- 笔记复制、重命名、删除（软删除 → 回收站）
- 笔记拖拽移动分类

### 2. 分类管理
- 无限层级分类树（parent_id 递归）
- 分类拖拽排序、拖拽调整层级
- 分类收藏
- 分类导出（JSON / Markdown / ZIP 含图片）
- 分类右键菜单（新建子文件夹、重命名、删除）
- 分类显示直接笔记数量（可在设置中开关）
- 笔记列表显示子文件夹，可点击进入导航

### 3. 标签管理
- 多标签关联（笔记-标签多对多）
- 标签颜色自定义
- 标签筛选（支持组合分类筛选）
- 标签右键菜单（重命名、更换颜色、删除）

### 4. 搜索
- 全局搜索（Ctrl+Shift+F）
- 搜索历史记录
- 搜索结果高亮
- 支持按标题 + 内容搜索

### 5. 日期归档
- 按年月分组显示笔记
- 侧边栏年月折叠导航

### 6. 回收站
- 软删除笔记
- 恢复 / 永久删除
- 清空回收站

### 7. 数据统计
- 笔记总数、本月新增、今日新增
- 分类 / 标签数量统计
- ECharts 图表可视化（分类分布饼图、月度趋势折线图）

### 8. 文件空间
- 管理笔记中引用的图片/附件
- 按分类分组显示
- 图片预览
- 未引用文件清理
- 批量删除

### 9. 外部文件浏览器
- 导入外部文件夹到侧边栏
- 树形目录结构（懒加载）
- 支持预览 Markdown / 文本 / JSON 文件
- 支持编辑并保存
- 拖拽文件到分类可导入

### 10. 数据导入导出
- JSON 完整备份 / 恢复
- Markdown 批量导出（ZIP 含图片）
- Markdown 批量导入
- 单笔记导出（MD / HTML / ZIP）
- 分类导出为 ZIP

### 11. 云同步
- SSH/SFTP 连接（支持密码和私钥）
- 上传本地数据库 + 图片到云端
- 从云端下载覆盖本地
- 连接测试

### 12. 应用设置
- 深色模式切换
- 关闭行为（直接退出 / 最小化到托盘）
- 图片存储模式（Base64 / 文件夹）
- 显示文件夹内文件数量开关
- 数据库路径查看
- 全局快捷键

## 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | HomeView | 首页（笔记列表 + 编辑器） |
| `/favorites` | FavoriteView | 收藏笔记 |
| `/tags` | TagView | 标签管理 |
| `/search` | SearchView | 搜索结果 |
| `/trash` | TrashView | 回收站 |
| `/stats` | StatsView | 数据统计 |
| `/files` | FileSpaceView | 文件空间 |
| `/external-preview` | ExternalFilePreview | 外部文件预览 |
| `/settings` | SettingsView | 设置 |

## 侧边栏结构

三段式可拉伸布局：
1. **顶部**：导航菜单 + 分类树 + 日期归档
2. **中间**：标签列表
3. **底部**：外部文件浏览器

分隔条可拖拽调整各区域高度。
